Ce projet est une création des parties front et back d'un blog en PHP et MySQL (architecture MVC, modèles et vue en POO).

Pour récupérer le répositoire,
- cliquer sur le bouton clone ou download 
- récupérer l'URL dans 'clone with HTTPS'
- dans la console de commande, se placer dans le dossier de son choix (cd) et entrer : 'git clone' suivi de l'URL.

Renommer le fichier 'Mangager-sample.php' en 'Manager.php'. En modifier les données de connexion 'id' et 'password'. Me demander :
- un export de la base de données et l'importer depuis phpMyAdmin.
- le dossier assets/images.

Télécharger JQuery 3.4.1 (https://jquery.com/download/). Placer le fichier jquery-3.4.1.min.js dans assets/javascript/.
